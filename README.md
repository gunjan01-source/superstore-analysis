# Retail Sales Performance Analysis

## Project Overview

This project analyzes retail sales data to uncover business insights using Python, SQL, and Power BI.

## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- SQL (MySQL)
- Power BI

## Objectives

- Analyze sales performance
- Identify profitable categories and regions
- Track monthly sales trends
- Build an interactive dashboard
